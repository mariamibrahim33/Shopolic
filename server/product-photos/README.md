# Add your own product photos here

1. Save each photo with **one of the exact names below** (any of `.jpg`, `.jpeg`,
   `.png`, `.webp` works — the extension doesn't matter, only the name does).
2. Drop the files into **this folder** (`server/product-photos/`).
3. From the `server/` folder, run:  **`npm run images`**
4. Refresh the app — the new photos appear.

You don't have to do all 14 at once. Add a few, run the command, add more later.
